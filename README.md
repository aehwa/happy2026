# 새해 인사 웹페이지

지인들에게 보낼 연하장 대신 사용할 웹페이지입니다.

## 사용 방법

### 1. 이미지 준비
- `images` 폴더를 생성하세요
- PNG 형식의 이미지 파일 20개를 준비하세요
- 파일명: `image1.png`, `image2.png`, ..., `image20.png` 형식으로 저장하세요

### 2. 텍스트 문구 준비
- `data.js` 파일을 열어주세요
- `texts` 배열에 50개의 문구를 추가하세요
- 예시:
```javascript
const texts = [
    "새해 복 많이 받으세요!",
    "건강하고 행복한 한 해 되세요!",
    // ... 나머지 문구들
];
```

### 3. 실행
- `index.html` 파일을 웹 브라우저로 열어주세요
- 또는 로컬 서버를 실행하세요:
  ```bash
  # Python 3
  python -m http.server 8000
  
  # Node.js (http-server 설치 필요)
  npx http-server
  ```

## 기능
- 50개의 문구 중 하나를 랜덤하게 표시
- 20개의 이미지 중 하나를 랜덤하게 표시
- "다시 보기" 버튼으로 새로운 조합 확인 가능
- 반응형 디자인 (모바일 지원)
